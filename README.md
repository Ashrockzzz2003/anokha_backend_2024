# Anokha 2024

Anokha 2024 TechFest server.

## Installation

1. Install Node.js and npm
2. Install mysql.
3. Create Databases.

```
CREATE DATABASE anokha;
CREATE DATABASE anokha_transactions;
```

4. Install dependencies.

```
npm i
```

5. Run the server.

```
npm start
```
